# MetaScan — Chrome Web Store Listing

## Extension Name

**MetaScan — SEO Meta Tags & Social Preview**

## Short Description (132 chars max)

Extract meta tags, preview social sharing, and audit SEO on any page. Free, instant, no signup.

## Long Description

MetaScan shows you exactly how your web pages appear in search results and social feeds — before your users see them.

**SCAN ANY PAGE**
Click the MetaScan icon on any webpage to instantly extract all meta tags, Open Graph properties, Twitter Cards, canonical URLs, robots directives, heading structure, and image alt text. Everything organized in one clean dashboard.

**SOCIAL PREVIEW THAT'S ACTUALLY ACCURATE**
See real-time previews of how your content renders on:
- Facebook / LinkedIn (Open Graph)
- X / Twitter (Twitter Cards)
- WhatsApp / Slack (link previews)

No more guessing whether your og:image is the right size or your meta description gets truncated.

**15+ SEO ISSUE CHECKS**
MetaScan automatically flags common problems that hurt your rankings:
- Missing or too-long title tags
- Missing, duplicate, or short meta descriptions
- Missing Open Graph or Twitter Card tags
- Broken canonical URLs
- Missing viewport meta tag
- Images without alt text
- Missing lang attribute on html element
- Heading hierarchy issues

Each issue comes with a plain-English explanation and fix suggestion.

**EDIT META TAGS IN-BROWSER**
Use the built-in Editor to modify page titles, descriptions, OG tags, Twitter Cards, and canonical URLs — all with live character counters that show you exactly when you're approaching platform limits.

**EXPORT REPORTS**
Download full page audits as:
- JSON — structured data for developers
- CSV — spreadsheet-compatible for teams
- HTML — printable reports for clients

**PRO TIER**
Upgrade to MetaScan Pro for advanced features including auto-scan on page load, custom issue rules, and priority support.

**PRIVACY-FIRST**
MetaScan runs entirely in your browser. No data is sent to external servers. No tracking. No accounts required.
