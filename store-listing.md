# MetaScan — Chrome Web Store Listing

## Extension Name

**MetaScan — SEO Meta Tags & Social Preview**

## Short Description (132 chars max)

Extract meta tags, preview social sharing, and audit SEO on any page. Free, instant, no signup.

## Long Description

MetaScan shows you exactly how your web pages appear in search results and social feeds — before your users see them. Click the MetaScan icon on any webpage to instantly extract all meta tags, Open Graph properties, Twitter Cards, canonical URLs, robots directives, heading structure, and image alt text.

**SCAN ANY PAGE**
Everything is organized in one clean dashboard: page title, meta description, Open Graph tags, Twitter Cards, canonical URL, robots meta, heading hierarchy, and image alt text. No more switching between multiple tools to audit a page's SEO metadata.

**SOCIAL PREVIEW THAT'S ACTUALLY ACCURATE**
See real-time previews of how your content renders on Facebook/LinkedIn (Open Graph), X/Twitter (Twitter Cards), and WhatsApp/Slack (link previews). No more guessing whether your `og:image` is the right size or your meta description gets truncated.

**15+ SEO ISSUE CHECKS**
MetaScan automatically flags common problems that hurt your rankings: missing or too-long title tags, missing/duplicate/short meta descriptions, missing OG/Twitter Card tags, broken canonical URLs, missing viewport meta, images without alt text, missing lang attribute, and heading hierarchy issues. Each issue comes with a plain-English explanation and fix suggestion.

**EDIT META TAGS IN-BROWSER**
Use the built-in Editor to modify page titles, descriptions, OG tags, Twitter Cards, and canonical URLs — all with live character counters that show you exactly when you're approaching platform limits.

**EXPORT REPORTS**
Download full page audits as JSON (structured data for developers), CSV (spreadsheet-compatible for teams), or HTML (printable reports for clients).

**PRO TIER**
Upgrade to MetaScan Pro for advanced features including auto-scan on page load, custom issue rules, and priority support.

**PRIVACY-FIRST**
MetaScan runs entirely in your browser. No data is sent to external servers. No tracking. No accounts required.

## Key Features

- **Instant meta tag extraction** — all meta, OG, Twitter Cards, canonical, robots, headings, images
- **Social preview rendering** — Facebook/LinkedIn, X/Twitter, WhatsApp/Slack
- **15+ SEO issue checks** — missing title, short description, missing OG tags, and more
- **In-browser meta tag editor** — with live character counters
- **Multi-format export** — JSON, CSV, HTML
- **Pro tier** — auto-scan, custom rules, priority support
- **Zero dependencies** — pure vanilla JavaScript, no frameworks
- **Privacy-first** — all processing is local, no tracking

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `metascan/` directory
5. The MetaScan icon appears in your toolbar

## Requirements

- Chrome 88+ (Manifest V3 support)
- No additional dependencies
